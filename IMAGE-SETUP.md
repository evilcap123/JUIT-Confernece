# Image Placement Instructions

## Required Images

You need to place the following images in your assets folder:

### 1. Hero Banner (Image 1)

- **File name:** `hero-banner.png`
- **Location:** `assets/img/hero-banner.png`
- **Source:** The blue gradient conference banner image you provided
- **Usage:** Background image for the hero section at the top of the page

### 2. Group Photo (Image 2)

- **File name:** `group-photo.jpg`
- **Location:** `assets/img/group-photo.jpg`
- **Source:** The JUIT Civil Engineering Department group photo
- **Usage:** First item in the Events Gallery section

## How to Add the Images

1. Save/download your two images
2. Rename them as specified above
3. Copy them to the `assets/img/` folder in your project

## Current Status

The HTML has been updated to reference these images. Once you place the files in the correct location, they will automatically display on your website.

Run `npm run dev` to see the changes in real-time as you add the images.
